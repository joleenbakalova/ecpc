About this data

This repository continues the ECPC evidence review for the period after its 31 July 2023 cutoff. Each record links to a publisher page, open-access source, or the author's official profile so findings can be verified at the source.

Status badges. Peer-reviewed, Grey and Book denote output type. A Lead badge marks a strong lead whose exact date, venue, or DOI still needs manual confirmation — treat it as a to-do, not a finished record. Sources used: publisher sites, PubMed/PMC, institutional portals, ORCID, Google Scholar, and ecdpeace.org. The licensed databases in the original protocol (Web of Science, PsycINFO, ERIC, Medline, APA PsycArticles, SocINDEX) were not searched here.

Coverage so far

Authors fully or partly swept in this build: Aber, Yoshikawa, Connolly, Dunne, Miller, Betancourt. The remaining ECPC members are queued — the companion spreadsheet holds a full-list source link (ORCID / portal / Scholar) for all 46 so coverage can be completed and audited.

Built as a working prototype for the Early Childhood Peacebuilding Consortium evidence library. Findings quoted are extracted from the full text of each publication; open each source for complete results.
